AndroidAsync 是一个底层的基于网络协议，提供各种网络通讯服务类库。它功能强大，提供包括raw Socket, HTTP client/server, WebSocket, 以及 Socket.IO 的类库。AndroidAsync是基于NIO的。 AndroidAsync具体功能包括： 1. 基于NIO。一个线程由callback函数驱动。效率很高。 2. 所有的操作都返回一个Future（http://en.wikipedia.org/wiki/Futures_and_promises），并且可以取消； 3. Socket client + socket server 4. HTTP client + server 5. WebSocket client + server 6. Socket.IO client：
