File->Settings->Keymap

===================Editor Actions===================
删除行 : Delete Line -> Ctrl+d
复制行 ：Duplicate Line or Block -> Ctrl+Down


===================Main menu -> Code -> Completion===============
代码提示 ： Basic -> Shift + f

===================Main menu -> Code -> Folding===============
下移行 ： Move Line Down -> Alt+Down
上移行 ： Move Line Up -> Alt+Up

===================Main menu -> Navigate===============
查找类 : Class... -> Ctrl+Shift+T
查找文件 : File... -> Ctrl+Shift+R
显示当前文件的结构  : File Structure -> Ctrl + O 
显示类结构图  : Type Hierarchy -> Ctrl + T 

===================Main menu -> Edit -> Find===============
全局搜索 : Find in path... -> Ctrl+H



===================================================
导入包,自动修正(Show Intention Actions) : Alt+回车

看到当前方法的声明(显示当前光标在哪个方法里面)(Context Info) : Alt＋Q

生成代码(如get,set方法,构造函数等)(Generate...) : Alt+Insert

可以显示最近编辑的文件列表(Recent Files) : Ctrl＋E

关闭文件 : Shift＋Click

跳到大括号的开头结尾(Move Caret to Code Block Start) : Ctrl＋[或]

查询当前元素在当前文件中的引用，然后按F3可以选择(Find Usages in File) : Ctrl＋F7

可以显示参数信息(Parameter Info) : Ctrl＋P

把代码包在一块内，例如try/catch(Surround With...) : Ctrl＋Alt＋T

可以选择单词继而语句继而行继而函数(Select Word at Caret) : Ctrl＋W

跳转到上次编辑的地方(Last Edit Location) ：Ctrl＋Shift＋Backspace
